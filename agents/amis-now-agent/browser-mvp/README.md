# Browser-MVP ohne Node

Dieses Snippet laeuft direkt in Edge auf `https://sdw.allianz.de`. Es benoetigt keine Installation und kein Passwort im Code.

## Ablauf

1. In Edge bei `https://sdw.allianz.de` anmelden.
2. Auf der Startseite bleiben.
3. Taste `F12` druecken.
4. Tab `Konsole` oeffnen.
5. Inhalt von `person-create-mvp.js` komplett einfuegen.
6. Enter druecken.

Standardmaessig ist `RUN_SUBMIT = false`, damit nur gefuellt wird. Wenn alles korrekt aussieht, im Snippet oben `RUN_SUBMIT = true` setzen und erneut ausfuehren.
